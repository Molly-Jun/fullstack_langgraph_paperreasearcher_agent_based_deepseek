# RCAgent: Cloud Root Cause Analysis by Autonomous Agents with Tool-Augmented Large Language Models

## 摘要

本文提出RCAgent，一种面向工业云环境的工具增强型大语言模型（LLM）自主智能体框架，用于实现实用且隐私感知的根因分析（RCA）。该框架运行于内部部署模型，支持自由形式数据收集与综合分析，融合了动作轨迹自洽性（Self-Consistency）、上下文管理、稳定性增强及领域知识导入等多项改进。

RCAgent基于ReAct范式，包含两类LLM智能体：控制器智能体负责协调思考-行动-观察循环，专家智能体作为工具提供领域特定功能。为应对云环境中上下文长度限制、领域知识不足和动作有效性三大挑战，框架提出了四项增强方法：观察快照键（OBSK）机制以可控长度为智能体提供决策支持信息；语义极简设计的信息收集工具与基于Louvain社区检测的分析工具；JsonRegen与错误处理机制以提升动作稳定性；以及轨迹级自洽性（TSC）聚合方法。

实验在阿里云实时计算平台（Apache Flink）上进行，该平台高峰期吞吐量可达每秒1亿条数据记录。离线数据集包含161个经SRE团队标注的异常作业。RCAgent在根因、解决方案和证据预测三项子任务的所有指标上全面超越原始ReAct，根因和解决方案预测的METEOR分别提升+8.71和+6.52。消融实验验证了各组件（LLM专家、OBSK、JsonRegen）的有效性。在分布外（OoD）任务上，RCAgent获得2.92分（0-5 Likert量表），Tukey's HSD检验显示其显著优于ReAct（p=0.001），责任判定精确率达82.06%。资源消耗随数据量近似线性增长（Pearson相关p<0.05），性能无显著下降（Kruskal-Wallis H检验p=1.0）。

RCAgent已被集成到公司内部运维管理平台，专门分析现有自动SRE工具无法处理的域外任务，并将标记为平台责任的作业及其根因分析结果移交给Apache Flink的SRE团队，辅助人工诊断。该工作开创了LLM智能体在真实云RCA场景中的应用先河。

## 关键概念 / 实体

- **RCAgent**: 本文提出的工具增强型LLM自主智能体框架，用于云环境根因分析，通过内部部署模型确保工业级安全。
- **根因分析（RCA）**: 站点可靠性工程的核心组件，旨在从系统异常中定位根本原因，缩短平均解决时间。
- **ReAct**: 一种LLM智能体范式，通过交替进行推理（Reasoning）和行动（Acting）来完成任务，RCAgent在此基础上进行增强。
- **轨迹级自洽性（TSC）**: 一种聚合方法，仅在控制器智能体进入最终化阶段时开始采样，共享前期步骤以降低开销，平衡全程自洽性与单步思维链自洽性。
- **Apache Flink**: 阿里云实时计算平台的基础框架，RCAgent在该平台上进行开发、评估与部署，用于诊断异常流处理作业。
